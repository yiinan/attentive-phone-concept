# attentive-phone-concept

### Description
People pay a lot of attention on their phones. However, their eyes will get really tired after a day. This function can set a limitation for your eye time on the phone and inform you when the time is up. With this simple but emotional design, the users can feel that the phone takes care of them. It is also a good function for parents, who can set it for their children. I believe could be helpful for peopleâ€™s daily life.
